# Project Overview

| Project Name | What It Does | Technologies Used |
| --- | --- | --- |
| codeforge | General-purpose coding/project sandbox (utilities and samples). | Python, CLI tooling (typical), Git |
| vectordb-chroma | Vector database experiments and retrieval workflows. | Python, ChromaDB, Embeddings |
| ScholarRAG-Agentic-Research-Paper-Chatbot | Research paper RAG chatbot with agentic workflows. | Python, LLMs, RAG, Vector DB |
| LocalRAGStudio-Langchain | Local RAG application built with LangChain. | Python, LangChain, Vector DB |
| GraphScholar-RAG | Graph-based RAG for scholarly content. | Python, Graph DB, RAG |
| SlopeForge-House-Price-Regression-Lab | End-to-end regression pipeline for house prices with API serving. | Python, scikit-learn, FastAPI |
